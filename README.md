10. 应用
------------------------

有 x = (x0, x1, x2, ..., xn),

y = (y0, y1, y2, ..., yn).

令 distance = sqrt((x0-y0)**2 + (x1-y1)**2 + ... + (xn-yn)**2).

有 A = [ [x00, x01, ..., x0n],

        [x10, x11, ..., x1n],

        ...,

        [xm0, xm1, ..., xmn] ],
   
   y = [ y0, y1, ..., yn],

   求y与A中各元素的距离.
